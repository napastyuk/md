### Этап 3. BPMN и моделирование процессов
книга Брюс Сильвер — BPMN Method and Style
На русском полностью найти сложно, но даже переведенные конспекты очень полезны.
YouTube искать:
- BPMN для начинающих
- BPMN 2.0 за 2 часа
- Моделирование бизнес-процессов BPMN

Здесь важнее практика, чем теория, поэтому все пробуем в Draw.io / diagrams.net или любом другом редакторе BPMN. Что бы было что моделировать, можно записывать процессы из медицины:
- запись пациента;
- госпитализация;
- выписка;
- лабораторные исследования

### Этап 4 ПЕТ-проект
Заводим github или gitverse чтобы было куда выкладывать. Цель не выложить код, а показать навыки документирования моделирование процессов. Выбираем любой сложный процесс, можно опять медицинскую систему и описываем:
1. BPMN diagrams  
2. User stories  
3. Backlog
4. Database model (после этапа 5)
5. Requirements (после этапа 6)  
6. API description (после этапа 7)  

### Этап 5. SQL
Процесс поиска работы будет не быстрый, поэтому сразу после этапа 4 (ПЕТ-проект) начинаем откликаться на вакансии.  Как работать с вакансиями тема отдельной статьи. 
На этапе 5 надо Выучить ТОЛЬКО базовые операторы, не пытайся изучить весь SQL
