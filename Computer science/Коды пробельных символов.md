
| Символ | Пояснение                             |
| ------ | ------------------------------------- |
| ` `    | Пробел                                |
| `\r`   | Возврат каретки (Carriage return, CR) |
| `\n`   | Перевод строки (Line feed, LF)        |
| `\t`   | Табуляция (Tab))                      |
| `\v`   | Вертикальная табуляция (vertical tab) |
| `\f`   | Конец страницы (Form feed)            |
| `\b`   | Возврат на 1 символ (Backspace)       |
