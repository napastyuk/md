Допустим мы хотим найти строчку `moment.setLocale(` даже если ее уже перетёрли другими коммитами.

## Способ 1: Поиск по всем коммитам истории через `xargs`
```bash
git rev-list --all | xargs git grep "moment.setLocale("
```

## Способ 2: Поиск в **diff** (изменениях) каждого коммита
Флаг `-G` ищет именно по тексту изменений
Флаг `-p` Сразу покажет сам патч (код)
```bash
git log -G "moment.setLocale\(" -p
```
`\(` — скобку нужно заэкранировать, так как поиск идет по регулярному выражению.

## Способ 3: Если нужно только имя файла и коммит
```bash
git log -S "moment.setLocale(" --source --all
```
