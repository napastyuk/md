## Текстовые файлы
````bash
//Чтение

cat file1 //Отобразить все содержимое файла file1 начиная с первой строки

head -2 file1 //Отобразить две первые строки файла

//Создание 
touch newfile //создаст новый пустой файл

> newfile //делает тоже самое

echo blabla > newfile //создает новый пустой файл с текстом blabla
````
