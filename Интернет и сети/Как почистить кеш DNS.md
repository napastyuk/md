#сети #devops

DNS-кеш хранит IP-адреса посещённых сайтов локально, чтобы при каждом сетевом запросе не выполнять DNS-поиск.
DNS кеш содержит:
- Данные ресурса (или rdata)
- Тип записи
- Имя записи
- TTL (время жизни информации)
- Класс
- Длину данных ресурса
### Очистить DNS-кеш Chrome
1. Введите `chrome://net-internals/#DNS` в адресной строке
2. Кнопка _Clear Host Cache_ очистить DNS-кеш Chrome

Обычно не нужно но иногда может понадобится поменять в настройках Chrome
- Отключите **безопасный DNS**  `chrome://settings/security`
- Отключите **Async DNS resolver** `chrome://flags/#enable-async-dns`
- Очистить кеш socket `chrome://net-internals/#sockets`
### Очистить DNS-кеш Android

1 способ. Сменить тип wifi-сети на статическую и обратно
2 способ. сбросить сетевые настройки. Откройте _Настройки → Системные → Расширенные → Сброс → Сброс параметров сети_ и нажмите _Сбросить настройки_.
### Очистить DNS-кеш Windows
Запустите в cmd от администратора. 
`ipconfig /flushdns`
Вы увидите _Кеш DNS-резолвера успешно сброшен_
### Очистить DNS-кеш Linux
Сначала надо узнать, какой DNS-резолвер вы используете 
`sudo lsof -i :53 -S`
